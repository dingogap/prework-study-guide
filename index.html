HTML

  • The head element contains information about the web page.
  • Meta elements hold information affecting accessibility, performance, and for search engines.
  • The title element holds the web page name and is required.
  • The body element represents the visible content shown to the user.
  • The header element is at the top of the body element and holds menus and introductory content.
  • The main element contains information unique to the page.
  • The footer element appears at the bottom of the body and holds authro and copyright information as well as sitemap and contact information.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Changing the padding changes the size of the element

GIT

  • git status: checks what branch we are currently on
  • git pull origin main: pulls down the latest main branch
  • git checkout -b branch-name: creates a new branch and switches to it
  • git commit -m "comment": commits the changes made
  • all elements with the same class on a page will inherit the values of the properties defined for that class.

JavaScript

    • A variable is a named container that allows us to store data in our code.
    • Control flow is the order in which a computer executes code in a script.
    • Variables can be stored in arrays.
    • Arrays are zero indexed.
    • JavaScript can be debugged using the Console in Chrome DevTools.
